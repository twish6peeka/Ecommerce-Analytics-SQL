# E-commerce Analytics SQL Project

## Overview
This project builds an end-to-end analytical data model using SQL to analyze
revenue, customers, sellers, products, and operational performance.

## Key Features
- Staging-based ETL design
- Clean fact and dimension tables
- 26 business KPIs implemented as SQL views
- Revenue, customer, seller, and operational insights
- Geolocation analysis (optional, extensible)

## Tech Stack
- PostgreSQL
- DBeaver
- SQL

## Repository Structure
- sql/ → schema, ETL, KPIs
- data/ → sample raw data (LIMITED)
- kpis/ → sample KPI outputs
- docs/ → documentation

## Notes
This repository contains only sample data for demonstration purposes.
Full datasets are intentionally excluded.
